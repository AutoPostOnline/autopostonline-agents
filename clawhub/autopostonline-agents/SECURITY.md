# Security

This skill is docs-only.

It does not include:

- install scripts
- shell scripts
- dependency installers
- background processes
- browser automation
- local file access
- token extraction
- wallet access
- crypto functions
- system commands

The agent should never ask for social media passwords.

The owner connects social channels directly inside AutoPostOnline.

The agent receives only an AutoPostOnline API key.

The recommended default mode is safe mode:
draft first, publish after approval.

Autonomous mode should be used only after the owner explicitly authorizes the campaign scope, channels, posting frequency, and rules.
