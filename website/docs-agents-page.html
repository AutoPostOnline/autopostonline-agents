AutoPostOnline Agent Docs

AutoPostOnline is a smart, cheap, convenient social publishing layer for AI agents. Users connect social accounts in the dashboard. Agents publish through AutoPostOnline using an API key.

Endpoints

App:
https://app.autopostonline.com

API:
https://app.autopostonline.com/api

MCP:
https://app.autopostonline.com/api/mcp

Environment

POSTIZ_API_URL=https://app.autopostonline.com/api
POSTIZ_API_KEY=your_api_key

Authentication

Preferred:

Authorization: Bearer your_api_key

Fallback:

Authorization: your_api_key

First API test

curl -sS \
  -H "Authorization: Bearer $POSTIZ_API_KEY" \
  "$POSTIZ_API_URL/public/v1/integrations"

Recommended agent behavior

  1. List integrations before creating content.
  2. Create drafts before publishing.
  3. Ask approval before live publishing unless autonomous publishing was explicitly requested.
  4. Adapt copy per platform.
  5. Schedule campaigns when possible.
  6. Read analytics after publishing.

Pricing

Agent Unlimited: $9/month.

Built for agent builders who need reliable social publishing without high cost or complex setup.
